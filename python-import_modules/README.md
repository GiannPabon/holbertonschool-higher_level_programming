Python - Import & Modules

Description

This project covers the fundamental concepts of importing and using modules in Python. The exercises introduce importing functions from external files, utilizing built-in Python functions, creating and using modules, handling command-line arguments, and structuring Python programs effectively.

Learning Objectives

By the end of this project, you should be able to:

Understand why Python programming is powerful and widely used.

Import functions from external Python files.

Use imported functions efficiently.

Create and utilize modules.

Use the built-in dir() function.

Prevent code execution when a script is imported.

Handle command-line arguments in Python programs.

Requirements

Allowed editors: vi, vim, emacs

Environment: Ubuntu 22.04 LTS using Python 3.10.*

Code format: Follow pycodestyle (version 2.7.*)

Execution: All scripts must be executable.

Shebang: The first line of all scripts must be #!/usr/bin/python3

File endings: All files should end with a new line.

Project repository: holbertonschool-higher_level_programming

Project directory: python-import_modules

Project Tasks

0. Import a simple function from a simple file

Write a program that imports the function add(a, b) from add_0.py and prints the result of 1 + 2 = 3.

Use string formatting for output.

Ensure the script is not executed when imported.

1. My first toolbox!

Import mathematical functions from calculator_1.py.

Perform addition, subtraction, multiplication, and division.

Print results in a formatted manner.

2. How to make a script dynamic!

Write a program that prints the number of and the list of its arguments.

Output should display argument count and list the arguments with their position.

3. Infinite addition

Create a script that sums up all provided command-line arguments.

Handle big numbers efficiently.

4. Who are you?

Print all defined names in the compiled module hidden_4.pyc.

Print names in alphabetical order.

Exclude names starting with __.

5. Everything can be imported

Import and print the variable a from variable_load_5.py.

Ensure the script is not executed when imported.

Usage

To execute each script, run the following commands:

chmod +x script_name.py  # Ensure the script is executable
./script_name.py         # Run the script
