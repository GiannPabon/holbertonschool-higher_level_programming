Python - If/Else, Loops, and Functions

Description

This project covers fundamental control structures and functions in Python. You will learn how to use conditional statements, loops, and functions while following best practices in Python programming.

Learning Objectives

By the end of this project, you will be able to:

Understand the importance of indentation in Python.

Use if, if...else statements.

Write and use comments effectively.

Assign values to variables.

Use while and for loops.

Implement break and continue statements.

Use else clauses in loops.

Understand the purpose of the pass statement.

Work with the range() function.

Define and call functions.

Understand return statements and their behavior.

Recognize variable scopes.

Interpret traceback errors.

Use arithmetic operators in Python.

Requirements

All Python scripts must be executed with python3 (version 3.8.*) on Ubuntu 20.04 LTS.

The first line of each script must be #!/usr/bin/python3.

All code must follow pycodestyle (version 2.7.*).

All files must be executable.

A README.md file is required in the root directory of the project.

No external libraries should be used.

Tasks

Mandatory Tasks

Positive anything is better than negative nothing

Assigns a random number to a variable and prints whether it is positive, negative, or zero.

The last digit

Extracts and prints the last digit of a random number, followed by a condition-based message.

Alphabet game

Prints the lowercase alphabet without using additional variables.

Alphabet soup

Prints the alphabet except for q and e using a loop and a single print statement.

Hexadecimal printing

Prints numbers from 0 to 98 in both decimal and hexadecimal formats.

00...99

Prints numbers from 0 to 99, formatted with two digits and separated by commas.

Inventing is a combination of brains and materials

Prints all unique two-digit combinations in ascending order.

islower

A function that checks if a given character is lowercase.

To uppercase

A function that converts a string to uppercase without using built-in methods.

Last digit of a number

A function that prints and returns the last digit of a number.

a + b

A function that returns the sum of two numbers.

a ^ b

A function that computes the power of a raised to b.

FizzBuzz

Prints numbers from 1 to 100, replacing multiples of 3 with Fizz, 5 with Buzz, and both with FizzBuzz.
