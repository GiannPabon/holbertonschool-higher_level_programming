# Python - More Data Structures: Set, Dictionary

## Description
This project focuses on Python's more advanced data structures, specifically **sets** and **dictionaries**. The tasks will reinforce understanding of their usage, common methods, and when to use them over lists. Additionally, we explore lambda functions and functional programming methods like **map**, **filter**, and **reduce**.

---
## Learning Objectives
By the end of this project, you should be able to explain:
- Why Python programming is awesome.
- How to use **sets** and their common methods.
- When to use **sets** versus **lists**.
- How to iterate through a **set**.
- How to use **dictionaries** and their common methods.
- When to use **dictionaries** versus **lists** or **sets**.
- What a **key** is in a dictionary.
- How to iterate over a **dictionary**.
- What **lambda** functions are and how to use them.
- The purpose and functionality of **map, reduce, and filter** functions.

## Tasks

### 0. Squared simple
**File:** `0-square_matrix_simple.py`
- Function `square_matrix_simple(matrix=[])` computes the **square** of all integers in a 2D matrix without modifying the original matrix.
- Allowed to use loops or `map()`.

### 1. Search and replace
**File:** `1-search_replace.py`
- Function `search_replace(my_list, search, replace)` replaces all occurrences of `search` with `replace` in a new list.

### 2. Unique addition
**File:** `2-uniq_add.py`
- Function `uniq_add(my_list=[])` adds all **unique** integers in a list.

### 3. Present in both
**File:** `3-common_elements.py`
- Function `common_elements(set_1, set_2)` returns a **set** of common elements between two sets.

### 4. Only differents
**File:** `4-only_diff_elements.py`
- Function `only_diff_elements(set_1, set_2)` returns a **set** of elements found in only one of the two input sets.

### 5. Number of keys
**File:** `5-number_keys.py`
- Function `number_keys(a_dictionary)` returns the **number of keys** in a dictionary.

### 6. Print sorted dictionary
**File:** `6-print_sorted_dictionary.py`
- Function `print_sorted_dictionary(a_dictionary)` prints a dictionary sorted by keys in **alphabetical order**.

### 7. Update dictionary
**File:** `7-update_dictionary.py`
- Function `update_dictionary(a_dictionary, key, value)` updates or adds a key/value pair in a dictionary.

### 8. Simple delete by key
**File:** `8-simple_delete.py`
- Function `simple_delete(a_dictionary, key="")` deletes a key from a dictionary if it exists.

### 9. Multiply by 2
**File:** `9-multiply_by_2.py`
- Function `multiply_by_2(a_dictionary)` returns a new dictionary with all values **multiplied by 2**.

### 10. Best score
**File:** `10-best_score.py`
- Function `best_score(a_dictionary)` returns the **key** with the highest integer value in a dictionary.

### 11. Multiply by using map
**File:** `11-multiply_list_map.py`
- Function `multiply_list_map(my_list=[], number=0)` returns a list where each value is **multiplied by a number** using `map()`.
- No loops allowed, must be **at most 3 lines**.

### 12. Roman to Integer
**File:** `12-roman_to_int.py`
- Function `roman_to_int(roman_string)` converts a **Roman numeral** to an integer.
- Assumes numbers are between **1 and 3999**.
- Returns `0` if input is not a valid string.
